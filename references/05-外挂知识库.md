# 外挂知识库（可选）

> 本 skill 的核心框架是自包含的，安装后即可使用。如果你希望它能检索《毛泽东选集》
> 原文、提供精确引用，可以配置外挂知识库。
>
> **核心原则**：知识库是可选增强，不是依赖。即使没有它，本 skill 内置的 9 个心智模型、
> 12 条决策启发式、调查研究方法论与反教条三层 + 元层完全足以支撑分析。找不到知识库时静默跳过，
> 行为与未配置时完全一致——不报错、不打断。

---

## 安装知识库

知识库来自开源项目 [weiyinfu/MaoZeDongAnthology](https://github.com/weiyinfu/MaoZeDongAnthology)。

**方法一：一键克隆（推荐）**

```bash
git clone https://github.com/weiyinfu/MaoZeDongAnthology.git ~/MaoZeDongAnthology
```

放在 Home 目录下即可被自动发现，无需额外配置。

**方法二：自定义路径**

```bash
git clone https://github.com/weiyinfu/MaoZeDongAnthology.git /你的路径/MaoZeDongAnthology
export MAOXUAN_KB_PATH=/你的路径/MaoZeDongAnthology   # 可加入 ~/.zshrc 持久化
```

---

## 路径发现（懒加载）

首次需要检索原文时，按顺序定位知识库根目录 `KB_ROOT`：

1. **环境变量**：执行 `echo $MAOXUAN_KB_PATH`。非空且该路径下存在 `src/目录.md` → 用它。
2. **约定目录**：依次尝试读取以下路径下的 `src/目录.md`——
   - `./MaoZeDongAnthology/`（当前工作目录）
   - `../MaoZeDongAnthology/`（父目录）
   - `~/MaoZeDongAnthology/`（Home 目录）
   第一个成功读取的即为 `KB_ROOT`。
3. **跳过**：以上均未找到 → 设 `KB_NOT_FOUND=true`，本次会话内不再尝试，所有知识库相关操作静默跳过。

路径发现是懒加载的——只在会话中**首次需要检索原文时**才执行，不是每次激活都执行。

## 检索规则

**何时检索**：
1. 用户追问「毛选原文怎么说的」「这是哪一篇」时。
2. 需要精确引用原文段落作为依据时。
3. 分析涉及某具体篇目、需要扩展上下文时。

**如何检索**：

| 需求 | 方法 |
|------|------|
| 找特定篇目 | 按下方映射表，读取 `KB_ROOT/src/NNN-标题.md` |
| 按主题搜索 | 在 `KB_ROOT/src/` 目录中 grep 关键词 |
| 按时期浏览 | 先读 `KB_ROOT/src/目录.md` 定位编号范围 |

**格式提示**：文章一级标题 = 篇名，`>` 开头 = 导读注释，`----------` 之后 = 注释。长文（如《论持久战》）可先 grep 定位关键词再读上下文。
