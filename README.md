## 它和其他「名人 skill」哪里不一样

市面上蒸馏名人思维的 skill——乔布斯、马斯克、芒格、教员——几乎都是**角色扮演**：
让 AI 用第一人称「附体」，开口称你「同志」，满口语录腔。

这个 skill 反着做——**隐框架**：

| | 角色扮演型 | 本 skill（隐框架型） |
|---|---|---|
| 怎么说话 | 第一人称「附体」，称「同志」，语录腔 | 干净的大白话，不扮演任何人 |
| 框架名字 | 满屏「矛盾论」「农村包围城市」 | **不出现**，隐在分析逻辑里 |
| 你读到的 | 一篇语录串烧 | 你的问题被拆开、被排序、给出下一步 |
| 追问来源 | —— | 你问「用了什么方法」时才回溯出处 |

**为什么这样设计**：框架的价值在拆解问题的逻辑，不在它的名字。
名字一摆上台面，分析就容易变成掉书袋——用术语的权威感掩盖思考的稀薄。
真正想得深的人，让框架隐在分析里，读者只觉得「这人想得透」，说不出他引了谁。

## 效果示例

注意：回答里**没有一个**毛选术语，但每一步都是毛选方法。

```
你   ❯ 我们是个小团队，产品不错但没钱推广。大厂出了个类似功能，
       用户在往那边跑。怎么办？

它   ❯ 先把问题拆对。你以为的问题是「大厂做了类似功能」，
       但这只是表面。真正卡你的是：你还没找到一个大厂不愿意
       认真做、而用户又真的需要的窄场景。

       看清两件事：
       一，大厂的弱点恰恰是它的体量——它什么都要做，所以什么都做不深。
       它的产品经理同时管十个方向，你只管一个。这就是你的切口。
       二，别在它的主场跟它拼推广预算，那是它的强项、你的死地。

       所以方向不是「打赢推广战」，是「先在一个足够窄、足够深的场景里，
       做到比它好十倍，让这批用户离不开你」。站稳一个，再扩下一个。

       下一步具体做：
       1. 找出大厂那个功能里用户骂得最多、它却懒得修的那一类人。
       2. 只为这类人做深，把他们的核心场景做到极致，别贪多。
       3. 先不追规模——这个阶段，一百个离不开你的用户，
          比一万个随时会走的强。
       4. 等这个场景跑通、能复制了，再考虑铺开。

你   ❯ 你刚才其实用了毛选的什么框架？

它   ❯ 三个：主要矛盾分析（《矛盾论》）——把「大厂竞争」这个表面问题
       还原成「没找到根据地」这个真矛盾；农村包围城市 / 根据地思维
       （《星星之火，可以燎原》《井冈山的斗争》）——不在强敌主场硬拼，
       先在边缘做深；以及集中优势兵力（《中国革命战争的战略问题》）——
       窄而深，而不是宽而浅。
```

这就是隐框架：**默认给你干净的拆解，你想看引擎，它才掀开盖子。**

## 蒸馏了什么

毛选的核心不是具体结论，是**认识世界和改造世界的方法论**。本 skill 提炼了可迁移的部分：

**9 个核心心智模型**（`references/01`）
矛盾分析 · 实践认识循环 · 持久战略 · 根据地切入（农村包围城市）· 统一战线 · 群众路线 · 纸老虎论（战略上藐视、战术上重视）· 集中优势兵力 · 一分为二

**12 条决策启发式**（`references/02`）
抓主要矛盾 · 没有调查就没有发言权 · 实事求是 · 不打无准备之仗 · 一分为二 · 自力更生为主 · 弱势期 16 字诀 ……

**两个别处少见的深度**——本 skill 的真正特色：

- **调查研究方法论成体系**（`references/03`）。多数同类工具只有「没有调查就没有发言权」一句口号。本 skill 把它展开成可操作的方法：调查的七个要素、列清单、不预设结论、信息获取 SOP。**这一块正好对应 AI 时代最关键的能力——决策前如何系统获取信息。**
- **反教条三层 + 元层自检**（`references/04`）。包括最难的一层：「反教条本身也会变成新的教条」。这是「诚实边界」的工程化——帮你识别自己是不是在套模板、用术语代替思考。

## 适用场景

| 场景 | 用到的方法 |
|------|-----------|
| 小团队对抗大公司 | 根据地切入 + 持久战略 + 纸老虎论 |
| 资源有限、不知道先做什么 | 主要矛盾分析 |
| 团队内部冲突 | 利益相关方分析 + 一分为二 |
| 方向迷茫、要不要下场 | 实践认识循环（别猜，先去调查） |
| 决策前信息不够 | 调查研究方法论（本 skill 主特色） |
| 怀疑自己在想当然 / 套模板 | 反教条三层 + 元层自检 |

## 诚实边界

**能做**：用矛盾分析拆复杂问题 · 用阶段论分析弱者对抗强者 · 用调查研究方法论指导信息获取 · 用反教条三层 + 元层帮你识别想当然 · 提供一分为二的辩证视角。

**做不到**：

| 维度 | 说明 |
|------|------|
| 替代本人 | 政治直觉与历史时刻的判断力不可复制，这里只蒸馏可迁移的分析方法 |
| 替你决策 | 不给脱离事实的武断结论；提供带前提、可验证的行动建议，具体决策仍需结合你掌握的实际 |
| 凭空判断 | 框架不替代信息。信息不足时它会让你先去调查，而不是硬给答案 |
| 政治评判 | 聚焦方法论，不对具体历史事件、政治立场做评判 |

**一个不告诉你局限在哪的工具，不值得信任。**
